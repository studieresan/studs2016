studs2016
=========

Requirements
------------
* [node.js]
* [Bower]
* [mongoDB]

Installation
------------
`mv .env.example .env` and replace values.

```
npm install
bower install
node server
```

Workflow
--------
```
git checkout develop
git pull
git add [files]
git commit [message]
git push
```

[node.js]: <https://nodejs.org/en/>
[Bower]:   <http://bower.io/>
[mongoDB]: <https://www.mongodb.org/>
